# Depression Prediction System

An AI-powered Depression Prediction System that uses Deep Learning and Facial Emotion Recognition to analyze emotions from facial expressions and estimate depression risk levels. Built using TensorFlow, Keras, OpenCV, and the FER2013 dataset.

## Files
- MINI_PROJECT.ipynb - Training and prediction notebook
- best_emotion_model.h5 - Best trained model
- emotion_recognition_final.h5 - Final trained model

## Technologies
Python, TensorFlow, Keras, OpenCV, NumPy, Scikit-learn
