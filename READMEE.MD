# 🧠 Depression Detection System

## 📌 Overview
The Depression Detection System is an AI-powered application that uses Deep Learning and Convolutional Neural Networks (CNN) to detect potential signs of depression from facial expressions. The system analyzes facial images through preprocessing, feature extraction, and emotion classification to assist in early depression screening.

---

## 🌟 Features
- Facial emotion recognition using CNN
- Image preprocessing and normalization
- Feature extraction from facial images
- Depression prediction with over 82% accuracy
- User-friendly prediction interface
- Fast and efficient image classification
